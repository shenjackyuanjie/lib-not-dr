### lndl-nuitka

- [x] 修改配置路径
  - 从 `[tool.lndl.nuitka]` 变为 `[tool.lndl.nuitka.cli]`
- [x] 添加新配置
  - 添加 `[tool.lndl.nuitka.script]`
    - 用于项目动态修改 nuitka 脚本中的信息
    - 例如 `file-version` `product-version` 等

- 重构了一部分解析逻辑
  - 现在配置路径位于 `[tool.lndl.nuitka.cli]`
  - 可以添加脚本用于动态解析依赖
    - `[tool.lndl.nuitka]`
      - `script = "script.py"`
